# Sect Anchor Table

27 sects, each with a 7-dimension anchor point (0–100). Used by Phase 3.5 formula screening.

**Reading the table:**
- Anchor = 50 → this dimension is irrelevant to the sect's identity (weight 0.5 in matching)
- Anchor ≠ 50 → defining trait, the further from 50 the stronger the signal (weight 2.0 in matching)
- `core` column lists dimensions where anchor ≠ 50, i.e. the sect's personality signature

**Dimension keys:** 内力 neili · 轻功 qinggong · 招式 zhaoshi · 杀气 shaqi · 悟性 wuxing · 野心 yexin · 阅历 yueli
